# 中柬离线语音翻译 APP - 项目方案

## 一、项目目标

开发一款支持**离线语音翻译**的中柬翻译 APP，主要面向柬埔寨地区网络不稳定的使用场景。

## 二、核心功能

1. **离线语音识别**：支持中文和高棉语语音转文字
2. **离线翻译**：中柬双语互译
3. **离线语音合成**：朗读翻译结果
4. **常用短语库**：内置 100+ 常用短语

## 三、现有资源

### 3.1 高棉语语音数据集

| 数据集 | 规模 | 用途 |
|--------|------|------|
| fleurs_openslr42_mpwt | 5,778 条（3GB） | 训练集 |
| DDD-Cambodia/khmer-speech-dataset | 10万-100万条 | 大规模训练 |
| openslr-slr42-km-KH | 2,799 条 | 补充训练 |

**下载地址**：
- https://huggingface.co/datasets/KrorngAI/fleurs_openslr42_mpwt
- https://huggingface.co/datasets/DDD-Cambodia/khmer-speech-dataset

### 3.2 基础模型

- **OpenAI Whisper**：支持多语言语音识别，可微调
- **Vosk**：轻量级离线语音识别，适合手机端
- **Wav2Vec2**：Meta 开源的语音识别模型

## 四、技术方案

### 4.1 语音识别模型训练

```
步骤：
1. 下载高棉语语音数据集
2. 使用 Whisper-small 作为基础模型
3. 在 GPU 服务器上微调（约 20-50 小时训练）
4. 导出 ONNX 格式用于手机端部署
```

### 4.2 翻译模型

```
方案 A：离线词典 + 规则
- 构建中柬双语词典（10万+ 词条）
- 使用规则引擎进行简单翻译

方案 B：小型翻译模型
- 使用 MarianMT 或 NLLB 模型
- 量化后部署到手机端
```

### 4.3 语音合成

- 使用 Android 系统自带 TTS
- 或集成离线 TTS 模型

## 七、数据收集方案

### 7.1 现有数据集

直接使用 Hugging Face 上的开源数据集。

### 7.2 补充数据收集

**来源**：
1. **抖音/TikTok**：柬埔寨网红视频
2. **YouTube**：柬埔寨新闻、播客
3. **柬埔寨广播电台**：在线录音
4. **本地录制**：雇佣柬埔寨人朗读文本

**工具**：
```python
# 下载抖音视频
yt-dlp https://www.douyin.com/video/xxx

# 提取音频
ffmpeg -i video.mp4 -vn -acodec pcm_s16le -ar 16000 audio.wav

# 使用 Whisper 自动标注
whisper audio.wav --language km --task transcribe
```

## 十、风险与挑战

1. **数据质量**：高棉语数据可能不够多样化
2. **识别准确率**：小语种识别准确率可能不如大语种
3. **模型大小**：离线模型可能较大（100-200MB）
4. **口音问题**：柬埔寨不同地区口音差异
